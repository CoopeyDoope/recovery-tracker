# Recovery Tracker

A web-based application for tracking chest pain recovery exercises and progress.

## Access the App

[Open Recovery Tracker](https://coopeydoope.github.io/recovery-tracker/)

## Features

- Daily exercise tracking
- Pain level monitoring
- Google Sheets integration for data storage
- Progress visualization

## Usage

Click the link above to access the application. The tracker syncs with Google Sheets to store your daily exercise completions and pain assessments.
